# JUMBAK | جمبك

JUMBAK is a modern rickshaw ride-hailing app for Rufaa, Sudan.

## Goal
Provide a simple Uber-style experience adapted for local rickshaw transport: clear fares, nearby drivers, trip status, and an admin dashboard.

## Structure

- apps/mobile: passenger and driver mobile app
- apps/admin: admin dashboard
- server: API server
- server/prisma: database schema
- docs: project documentation
